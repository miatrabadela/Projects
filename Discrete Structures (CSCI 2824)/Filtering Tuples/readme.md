Instructions: You are given a list of tuples representing students and the courses they are enrolled in. Each tuple contains a student's name and a course code.

Design a Python function get_students_taking_at_least_one_course(students_courses, target_courses) that uses predicates and quantifiers to find students who are enrolled in at least one of the specified courses, and return their names in sorted order.
