Determines if an integer is a power of 7 by iteratively dividing out factors of 7 and checking if the remainder is 1. Demonstrates modulo arithmetic, integer division, and mathematical reasoning about prime factorization.

Instructions: For a given integer n, write a Python function isPowerOfSeven(n) that checks if the given integer n is a power of 7.
The function should return True if there exists an integer x ≥ 0 such that: n = 7^x
Otherwise, it should return False.
Assume that n is a non-negative integer (n ≥ 0).
